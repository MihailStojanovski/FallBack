Username and Password Generation

-Path of executable:
UserGeneration\bin\Debug\UserGeneration.exe

-Path of Users:
UserGeneration\bin\Debug\ReadingFrom\Users.txt

-Modes:

0) To obtain a PDF file with all of the users and their 
information on it call the command:
UserGeneration.exe 0

1) To obtain multiple PDF files with one users and their 
information on each call the command:
UserGeneration.exe 1

The default mode for the application (with no console
argument) is mode 0)
